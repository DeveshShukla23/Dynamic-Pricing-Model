> a self-created synthetic dataset.

---

## 📌 Project Overview

Dynamic pricing is a strategy where prices are adjusted in real-time
based on demand, competition, and market conditions. This project
builds and compares 3 ML models to find the best approach for
predicting optimal prices — complete with residual analysis,
data visualization and actionable business insights.

---

## 🤖 Models Compared

| Model | Type | Purpose |
|---|---|---|
| 📈 Linear Regression | Parametric | Baseline price prediction |
| 🌳 Decision Tree | Non-Parametric | Rule-based price segmentation |
| 🌲 Random Forest | Ensemble | High accuracy price prediction |

---

## 🔍 Project Features

- 📊 **Self-Created Synthetic Dataset** — Original data built in Python
- 📉 **Residual Visualization** — Error analysis for each model
- 📈 **Data Visualization** — Full visual pipeline throughout
- 🏅 **Model Accuracy Comparison** — Scores compared side by side
- 💼 **Business Insights** — Real-world pricing recommendations
- 🔮 **Future Suggestions** — Next steps for model improvement

---

## 📊 Model Performance Comparison

| Model | Strength | Best Used For |
|---|---|---|
| Linear Regression | Simple & interpretable | Linear price relationships |
| Decision Tree | Easy to visualize | Rule-based pricing segments |
| Random Forest | Highest accuracy | Complex pricing patterns |

---

## 💡 Business Insights

- 🌲 Random Forest outperforms other models in pricing accuracy
- 🌳 Decision Tree provides the most interpretable pricing rules
- 📈 Linear Regression serves as a strong baseline benchmark
- 💰 Ensemble methods are recommended for real-world pricing
- 🔮 Future scope includes adding demand elasticity features
